# Resource Reserver

Resource Reserver is a scheduling and reservation system for shared resources such as rooms, equipment, and vehicles. It provides a web UI, a REST API, and a CLI.

## What you can do

- Manage a catalog of resources with tags and availability status
- Book single or recurring reservations
- Join waitlists for full time slots
- Define business hours and blackout dates
- Subscribe to a personal calendar feed (iCal)
- Receive notifications and review audit history
- Integrate with external systems using webhooks
- Manage access with roles, MFA, and OAuth2
- Explore analytics and usage summaries
